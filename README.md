# Telegram Bot: AutoSale (Каталог автомобілів)

Асинхронний Telegram-бот для продажу автомобілів. Дозволяє користувачам переглядати каталог з пагінацією та шукати авто за детальними фільтрами. Включає захищену панель адміністратора для керування базою даних безпосередньо через інтерфейс Telegram.

## 🛠 Технологічний стек
* **Мова:** Python 3.10+
* **Фреймворк:** aiogram 3.x
* **База даних:** SQLite
* **ORM:** SQLAlchemy (Async)
* **Керування станами:** FSM (Finite State Machine)

## 🌟 Основний функціонал

### Для клієнтів:
* **Каталог авто:** Перегляд автомобілів у форматі каруселі з Inline-кнопками (Вперед/Назад) та фотографіями.
* **Система фільтрації:** Пошук за маркою, роком випуску (від і до), ціновим діапазоном, типом пального та коробкою передач.
* **Інтеграція з каналом:** Можливість переходу за посиланням на детальний пост у Telegram-каналі.
* **Контакти:** Швидкий доступ до інформації про автомайданчик.

### Для адміністраторів (доступ за ID):
* **Додавання авто:** Покрокове опитування (FSM) для завантаження характеристик, фотографії та посилання на пост.
* **Видалення авто:** Кнопка видалення, яка з'являється в каталозі виключно для авторизованих адміністраторів.

## 🚀 Інструкція із запуску (Локально)

1. **Клонуйте репозиторій:**
    git clone https://github.com/ВАШ_ЮЗЕРНЕЙМ/ВАШ_РЕПОЗИТОРІЙ.git
    cd ВАШ_РЕПОЗИТОРІЙ

2. **Створіть та активуйте віртуальне середовище:**
    python -m venv venv
    # Для Windows:
    venv\Scripts\activate
    # Для Linux/Mac:
    source venv/bin/activate

3. **Встановіть залежності:**
    pip install -r requirements.txt

4. **Налаштуйте змінні середовища:**
    Скопіюйте файл .env.example та перейменуйте його на .env.
    Відкрийте .env і додайте свій токен бота (від @BotFather) та ваш Telegram ID:
    
    BOT_TOKEN=ваш_токен
    ADMIN_IDS=ваш_ідентифікатор

5. **Запустіть бота:**
    python main.py

## 📂 Структура проєкту

* main.py — Точка входу, ініціалізація бота та диспетчера.
* config.py — Завантаження конфігурації з .env.
* database/ — Моделі SQLAlchemy (models.py) та функції для роботи з БД (requests.py).
* handlers/ — Обробники повідомлень та колбеків (каталог, фільтри, адмінка, контакти).
* keyboards/ — Генератори Reply та Inline клавіатур.
* states/ — Класи FSM для збереження станів користувача.
* filters/ — Кастомні фільтри (наприклад, перевірка прав адміністратора).
